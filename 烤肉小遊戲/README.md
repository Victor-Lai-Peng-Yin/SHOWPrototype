# 烤肉小遊戲

這是一款類似魔物獵人烤肉的小遊戲，玩家需要在正確的時機按下按鈕來控制烤肉的火候，達到目標熟度。

## 遊戲特色

- 🎯 **隨機目標熟度**：每次遊戲會隨機出現 3分、5分、7分 三種目標
- 🤖 **AI對手**：兩個AI對手同時進行烤肉，增加競爭性
- 🎮 **廚藝系統**：廚藝數值越高，遊戲難度越低
- 📊 **詳細結果**：顯示分數、排名和時間誤差

## 難度系統說明

### 廚藝數值影響

廚藝數值（1-100）會影響以下幾個方面：

1. **Timing 窗口大小**
   - 廚藝 1：窗口時間約 0.5秒（非常困難）
   - 廚藝 50：窗口時間約 1.25秒（中等難度）
   - 廚藝 100：窗口時間約 2.0秒（非常簡單）

2. **視覺提示**
   - 低廚藝（1-30）：基本顏色變化
   - 中廚藝（31-70）：顏色變化 + 進度條 + 聲音提示
   - 高廚藝（71-100）：完整視覺提示 + 聲音 + 震動

3. **速度穩定性**
   - 廚藝越高，烤肉速度變化越平緩，更容易掌握節奏

4. **分數加成**
   - 廚藝數值越高，即使不是完美時機，也能獲得更好的分數加成

5. **AI對手難度**
   - AI對手的表現會根據玩家廚藝數值調整，保持遊戲挑戰性

## 如何使用

### 方法一：直接在瀏覽器打開

1. 確保所有文件都在同一個資料夾中
2. 使用現代瀏覽器（Chrome、Firefox、Edge等）打開 `index.html`
3. 調整廚藝數值滑桿
4. 點擊「開始遊戲」
5. 在正確時機按下「按下！」按鈕

### 方法二：使用本地服務器

如果遇到模組載入問題，可以使用本地服務器：

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然後在瀏覽器中訪問 `http://localhost:8000`

## 遊戲玩法

1. **準備階段**
   - 調整廚藝數值（1-100）
   - 點擊「開始遊戲」

2. **倒數階段**
   - 等待倒數計時結束（3秒）

3. **遊戲階段**
   - 觀察進度條移動
   - 黃色標記表示完美時機
   - 綠色區域表示窗口時間範圍
   - 在正確時機按下「按下！」按鈕

4. **結果階段**
   - 查看你的分數和排名
   - 查看與AI對手的比較

## 分數判定

- **完美（PERFECT）**：時間誤差 < 窗口時間的 10%
- **良好（GOOD）**：時間誤差 < 窗口時間的 30%
- **普通（NORMAL）**：時間誤差 < 窗口時間的 50%
- **失敗（FAIL）**：時間誤差 > 窗口時間的 50%
